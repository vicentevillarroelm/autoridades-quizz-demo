
# Quiz de Autoridades de Chile 🇨🇱

Este es un proyecto React que te permite practicar el reconocimiento de autoridades chilenas.

## 🚀 Cómo usar

1. Asegúrate de tener instalado Node.js.
2. Descomprime el archivo.
3. Abre una terminal en la carpeta del proyecto.
4. Ejecuta:

```bash
npm install
npm start
```

¡Listo! Ya puedes jugar con el quiz.

## 📂 Reemplazar imágenes

Las imágenes están en `/public/images`. Puedes reemplazarlas por fotos reales manteniendo los nombres (image1.jpg, image2.jpg, etc).
